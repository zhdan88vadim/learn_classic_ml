Если вы меняете функцию потерь (Loss), то **меняются и формулы для градиентов** (`dw` и `db`), потому что вы теперь берете производную **от другой функции**.

---

## Как это работает

**Общий принцип:**
1. Вы выбираете функцию потерь \( L(y_{pred}, y_{true}) \)
2. Вычисляете её производную по \( y_{pred} \): \( \frac{\partial L}{\partial y_{pred}} \)
3. Умножаете на производную \( y_{pred} \) по \( w \) и \( b \) (это и есть **цепное правило**):
   - \( \frac{\partial y_{pred}}{\partial w} = x \)
   - \( \frac{\partial y_{pred}}{\partial b} = 1 \)

---

## Пример 1: MSE (то, что у вас сейчас)

**Функция потерь:**
\[
L = \frac{1}{m} \sum_{i=1}^{m} (y_{pred}^{(i)} - y_{true}^{(i)})^2
\]

**Производная по \( y_{pred} \):**
\[
\frac{\partial L}{\partial y_{pred}} = \frac{2}{m} \cdot (y_{pred} - y_{true})
\]

**Градиенты (через цепное правило):**
\[
\frac{\partial L}{\partial w} = \frac{\partial L}{\partial y_{pred}} \cdot \frac{\partial y_{pred}}{\partial w} = \frac{2}{m} \sum_{i=1}^{m} (y_{pred}^{(i)} - y_{true}^{(i)}) \cdot x^{(i)}
\]
\[
\frac{\partial L}{\partial b} = \frac{\partial L}{\partial y_{pred}} \cdot \frac{\partial y_{pred}}{\partial b} = \frac{2}{m} \sum_{i=1}^{m} (y_{pred}^{(i)} - y_{true}^{(i)})
\]

**В коде:**
```python
dw = (2 / m) * np.dot(X.T, (y_pred - y))
db = (2 / m) * np.sum(y_pred - y)
```

---

## Пример 2: MAE (Mean Absolute Error)

**Функция потерь:**
\[
L = \frac{1}{m} \sum_{i=1}^{m} |y_{pred}^{(i)} - y_{true}^{(i)}|
\]

**Производная по \( y_{pred} \):**
\[
\frac{\partial L}{\partial y_{pred}} = \frac{1}{m} \cdot \text{sign}(y_{pred} - y_{true})
\]
где `sign(x) = 1`, если `x > 0`, `-1`, если `x < 0`, и `0`, если `x = 0`.

**Градиенты (через цепное правило):**
\[
\frac{\partial L}{\partial w} = \frac{1}{m} \sum_{i=1}^{m} \text{sign}(y_{pred}^{(i)} - y_{true}^{(i)}) \cdot x^{(i)}
\]
\[
\frac{\partial L}{\partial b} = \frac{1}{m} \sum_{i=1}^{m} \text{sign}(y_{pred}^{(i)} - y_{true}^{(i)})
\]

**В коде:**
```python
dw = (1 / m) * np.dot(X.T, np.sign(y_pred - y))
db = (1 / m) * np.sum(np.sign(y_pred - y))
```

---

## Пример 3: Huber Loss (комбинация MSE и MAE)

**Функция потерь (для одного примера):**
\[
L = 
\begin{cases}
\frac{1}{2} (y_{pred} - y_{true})^2 & \text{если } |y_{pred} - y_{true}| \le \delta \\
\delta \cdot |y_{pred} - y_{true}| - \frac{1}{2}\delta^2 & \text{иначе}
\end{cases}
\]

**Производная по \( y_{pred} \):**
\[
\frac{\partial L}{\partial y_{pred}} = 
\begin{cases}
y_{pred} - y_{true} & \text{если } |y_{pred} - y_{true}| \le \delta \\
\delta \cdot \text{sign}(y_{pred} - y_{true}) & \text{иначе}
\end{cases}
\]

**В коде:**
```python
delta = 1.0
diff = y_pred - y
mask = np.abs(diff) <= delta
grad = np.where(mask, diff, delta * np.sign(diff))

dw = (1 / m) * np.dot(X.T, grad)
db = (1 / m) * np.sum(grad)
```

## Пример 4: Log-Cosh Loss (гладкая аппроксимация MAE)

**Функция потерь:**
\[
L = \frac{1}{m} \sum_{i=1}^{m} \log(\cosh(y_{pred}^{(i)} - y_{true}^{(i)}))
\]

**Производная по \( y_{pred} \):**
\[
\frac{\partial L}{\partial y_{pred}} = \frac{1}{m} \cdot \tanh(y_{pred} - y_{true})
\]

**В коде:**
```python
dw = (1 / m) * np.dot(X.T, np.tanh(y_pred - y))
db = (1 / m) * np.sum(np.tanh(y_pred - y))
```

## Общая схема для любой функции потерь

```python
def train_step(X, y, w, b, learning_rate):
    m = X.shape[0]
    
    # 1. Forward pass (всегда одинаковый)
    y_pred = np.dot(X, w) + b
    
    # 2. Вычисляем градиент функции потерь по y_pred
    # Это ЕДИНСТВЕННОЕ место, которое меняется!
    grad_loss = compute_gradient_loss(y_pred, y)  # ← разное для каждой Loss
    
    # 3. Цепное правило (всегда одинаковое!)
    dw = (1 / m) * np.dot(X.T, grad_loss)
    db = (1 / m) * np.sum(grad_loss)
    
    # 4. Обновляем веса (всегда одинаковое)
    w = w - learning_rate * dw
    b = b - learning_rate * db
    
    return w, b
```

## Сравнение разных Loss функций

| Функция потерь | Производная \( \frac{\partial L}{\partial y_{pred}} \) | Особенности |
|----------------|------------------------------------------------------|-------------|
| **MSE** | \( \frac{2}{m}(y_{pred} - y_{true}) \) | Чувствительна к выбросам, гладкая |
| **MAE** | \( \frac{1}{m}\text{sign}(y_{pred} - y_{true}) \) | Устойчива к выбросам, негладкая в 0 |
| **Huber** | \( \frac{1}{m} \cdot \text{clip}(y_{pred} - y_{true}, -\delta, \delta) \) | Комбинация MSE и MAE |
| **Log-Cosh** | \( \frac{1}{m}\tanh(y_{pred} - y_{true}) \) | Гладкая, устойчива к выбросам |

---

## Почему это важно понимать?

1. **Разные задачи требуют разных Loss:** 
   - Регрессия → MSE, MAE, Huber
   - Классификация → Cross-Entropy, Hinge Loss
   
2. **Выбор Loss влияет на обучение:**
   - MSE "наказывает" большие ошибки сильнее (квадрат)
   - MAE одинаково наказывает все ошибки
   - Huber — золотая середина

3. **Производная определяет, как обновляются веса:**
   - Если производная большая → веса меняются сильно
   - Если производная маленькая → веса меняются слабо
